Borders

The border property alaows you to specify the style, width, and color of an element's border.

Border Style

The border-style property can take up to four values (top border, right border, bottom border, and left border) The following border-style values and are allowed:

  • dotted: dotted border
  • dashed: dashed border
  • solid: solid border
  • double: doulble border
  • groove: 3D grooved border, and depends on the border-color value
  • ridge: 3D ridged border, and depends on the border-color value
  • inset: 3d inset border, and depedns on the border-color value
  • none: no border
  • hidden: hides the border
  • mixed: can set different border styles for the top, bottom, left, and right

Boder Color

VERY IMPORTANT: border-color will not work unless you have a border-style set!!

Border-color can be set by the following ways:

  • name:specify a colorr that is preset in HTML
  • HEX: specify a hex value, ie. "eeeeeee" or the shorthand "#eee" which works in some cases
  • RGB:specify a RGB value, ie. "rgb(100, 100, 100"
  • transparent will make the border color transparent
